Some comments at the beginning of the file.

As in C, these block comments can't be nested.

After all, we only clip them out with a simple regex before reading
any tokens.
*/

triangle_5 <- {1 + 2 + 3 + 4 + 5} ;
result <- {call triangle_5} ;

/* The current value of 'result' should be 0. */
print(result) ;

/*
This should print out 15.

*/
